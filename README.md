## Quick Start

### Setup

1. **Install uv**: `curl -LsSf https://astral-sh.uv/install.sh | sh`
2. **Clone repo**
3. **Sync Env**: `uv sync`
4. **Install Hooks**: `uv run pre-commit install`

### Commands

- **Run App**: `uv run afgl-run`
- **Test**: `uv run pytest`
- **Format**: `uv run ruff format .`
- **Lint**: `uv run ruff check . --fix`
